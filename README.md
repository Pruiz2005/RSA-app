RSA Cryptography Simulator

A web-based, interactive simulator designed to demonstrate and understand the RSA
cryptography algorithm. This application provides an educational approach to
understand key generation, encryption, and decryption processes

AI Development Process
The application was fully built using a combination of different AI tools, including Gemini, Claude and ChatGPT.
The development process was really interactive. Multiple prompts were used to generate the initial project structure, implement cryptographic functions and design the user interface. 
When one AI produced unexpected behaviors, the code and the requirements were passed to
another model. This multi-model strategy helped improve debugging efficiency,
feature refining and correction until the final optimal result was achieved.
